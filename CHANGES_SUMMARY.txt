================================================================================

1. animate-blur-in (0.4s)
   - Effect: Fade in with blur clearing
   - Keyframes: blur(8px) → blur(0) + opacity 0→1
   - Usage: Messages, content entering

2. animate-blur-out (0.3s)
   - Effect: Fade out with blur
   - Keyframes: blur(0) → blur(6px) + opacity 1→0
   - Usage: Modals, content exiting

3. animate-spin-blur (0.5s)
   - Effect: 180° rotate + blur + scale
   - Keyframes: rotate(0) blur(4px) scale(0.95) → rotate(180) blur(0) scale(1)
   - Usage: Special reveals, emphasis

4. animate-collapse-blur (0.35s)
   - Effect: Height collapse + blur
   - Keyframes: max-height 500px → 0 + blur increase
   - Usage: Closing expandable content

5. animate-expand-blur (0.35s)
   - Effect: Height expand from blur
   - Keyframes: max-height 0 → 500px + blur decrease
   - Usage: Opening expandable content

6. animate-pulse-blur (2s infinite)
   - Effect: Continuous subtle blur pulse
   - Keyframes: blur(0) ↔ blur(1px) at 50%
   - Usage: Thinking box while streaming

7. animate-glow-pulse (1.5s infinite)
   - Effect: Brightness + shadow pulse
   - Keyframes: brightness(1) shadow(0,0,10px) ↔ brightness(1.1) shadow(0,0,20px)

================================================================================
MAJOR BUG FIX
================================================================================

ISSUE: App stuck at bottom during AI response generation
USER IMPACT: Could not read previous messages while AI was generating

BEFORE:
```
if (bottomRef.current && !userHasScrolled.current) {
  bottomRef.current.scrollIntoView(...)
}
// Problem: Doesn't auto-scroll during loading, so jumps around
```

AFTER:
```
if (bottomRef.current) {
  if (!isLoading || !userHasScrolled.current) {
    bottomRef.current.scrollIntoView(...)
  }
}
// Solution: Smooth scroll during generation, respects user scroll position
```

BENEFITS:
✅ User has control over scroll during generation
✅ Can read previous messages while AI generates
✅ "Scroll to Bottom" button appears when needed
✅ Smooth, not jarring
